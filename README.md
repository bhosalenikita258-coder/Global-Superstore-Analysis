# Global Superstore Analysis

## Overview

This Power BI project analyzes the Global Superstore dataset to provide insights into sales, profit, customers, and shipping performance through interactive dashboards.

## File

* `Global Superstore Analysis.pbix`

## Features

* Sales and Profit Analysis
* Region-wise Performance
* Category & Sub-category Insights
* Customer Segment Analysis
* Interactive Filters and Slicers

## Tools Used

* Power BI Desktop
* Power Query
* DAX

## How to Use

Open the `.pbix` file in Power BI Desktop and explore the dashboard.
